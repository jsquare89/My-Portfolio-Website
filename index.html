Tiny House Land Share

Tiny House Land Share is a web application that aims to connect land owners and tiny house owners. This web application provides users a platform to post listings and/or search land for rent or tiny house owner listings.

This project was created by myself in 3 months using C# ASP.Net Core. My aim for this project was to create a usefull web applicaiton that I could eventually host online and that would help solve finding land for people of alternative living lifestyles. Through building this web application I have gained a stronger understanding of how to integrate basic search functionality, use EF Core for identity login and roles authorization, create a messaging system, use MVC pattern effectively, and sharpen my frontend skills.

  • C#
  • ASP.NET Core 6
  • MVC
  • Dependancy Injection
  • EF Core
  • MS SQL
  • LINQ
  • HTML/CSS
  • Visual Studio 2022
  • AutoMapper
