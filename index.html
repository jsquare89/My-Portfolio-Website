Tiny House Land Share

Tiny House Land Share is a web application that aims to connect land owners and tiny house owners. This web application provides users a platform to post listings and/or search land for rent or tiny house owner listings.

This project was created by myself using C# ASP.NET Core. My aim for this project was to create a usefull web applicaiton that I could eventually host online and that would help solve finding land for people of alternative living lifestyles. Through building this web application I have gained a stronger understanding with integrating basic search functionality, EF Core identity login authentication and roles authorization, LINQ for handling data queries, messaging systems, MVC pattern, and sharpened my frontend skills.

Online Store GraphQL API

A GraphQL API that serves data for a swim apparel storefront. This API interfaces with a Postgres database via a lightweight Dapper ORM. Created with C# and .NET Core 6 in Visual Studio 2022. GraphQL allows for a client to easily make use of a schema to query for the exact data needed and nothing more.

Project Highlights

  • Provides GraphQL endpoint to query and mutate data with full CRUD across multiple entities
  • Implements GraphQL with Hot Chocolate
  • GraphQL Voyager for viewing schema with relationship graph
  • Client authorization for mutations
  • Insomnia for testing the GraphQL endpoint
  • Docker for easy hosting of PostgreSQL service

Project Challenges

  • GraphQL learning curve
  • Handling resolvers correctly
  • Awareness of N+1 problem in fetching query dependancies. Implemented data loaders
